192.168.0.0/24     192.168.0.0/25
/24 和 /25 指的是 24 位子网掩码和 25 位子网掩码

子网掩码只有一个作用, 就是将某个 IP 地址划分成网络地址和主机地址两部分.

简单的说就是把一个网络分成几个.

在 192.168.0.0/24 的网络中, IP 地址 192.168.0.0 - 192.168.0.255 都在同一网段中,
即 192.168.0.1 和 192.168.0.254 可以不借助路由器互相连通.

在 192.168.0.0/25 的网络中, IP 地址被分为两个段,
即 192.168.0.0 - 192.168.0.128 和 192.168.0.129 - 192.168.0.254,
即 192.168.0.1 和 192.168.0.130 不在同一网络中, 需要路由设备才能互相连通.

23: 512 个 IP
24: 256 个 IP
25: 128 个 IP
26: 64  个 IP
27: 32  个 IP  (size: 2 ^ (32 - 27) = 32)
